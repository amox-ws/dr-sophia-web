# Dr. Cheirakis - Gynecology Practice Website

A professional medical website for Dr. Lefteris Cheirakis, a gynecologist in Athens, Greece.

## About

This website provides information about Dr. Cheirakis's medical practice, including:

- Gynecology services
- Assisted reproduction treatments
- Endoscopic surgery procedures
- Pregnancy care

## Technologies

- React
- TypeScript
- Tailwind CSS
- Vite

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
